# Message Layer Security (MLS) の node.jsにおける動作確認をしたい。

## step 1 コードのセキュリティを調べる
npm install ts-mls
して、このモジュールに悪意があるコードが含まれていないかチェックする。
npm auditも活用する。

## step 2 動作確認
この ts-mlsのサンプルを実装し、実際に動くか確認をする。
暗号化されたメッセージを実際に表示し、エポックのIDも表示する。


## step 3 スループット確認

テスト用のデータを1000件程度生成し、エンコードとデコーディングのスループットを評価する。

## step 4 多ユーザー確認

ユーザー数を1000人まで増やして、鍵を追加したときのエポック再計算に
どの程度の時間がかかるかを評価する。
