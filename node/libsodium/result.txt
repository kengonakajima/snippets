libsodium secretbox ベンチマーク結果
対象: 1MB英語テキスト1回、100文字英語テキスト10000回
鍵長: 128ビット、256ビット（crypto_secretboxの32バイト鍵要件に合わせて生成）

計測値 (ms):
- 128-bit key  : 1MB暗号化 3.158 / 1MB復号 2.770 / 100文字×暗号化 308.235 / 100文字×復号 19.864
- 256-bit key  : 1MB暗号化 2.850 / 1MB復号 2.747 / 100文字×暗号化 308.067 / 100文字×復号 14.750

メモ:
- 128ビット鍵は16バイト乱数を基にcrypto_generichashで32バイトに引き伸ばして使用（有効エントロピーは128ビット）。
- スクリプトは benchmark.js で実行コマンドは `node benchmark.js`。
