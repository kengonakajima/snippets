# iOS Safariで動く、文字起こしアプリのサンプル

iOS SafariでWebアプリを開いて、
マイク入力を開始するボタンを押すと、
マイクが起動して音声がサーバーに送られ、
サーバーでOpenAIのSTTを使ってリアルタイムに文字起こしをして
それをWebアプリ側に送って、WebアプリでDOMに反映していくだけのサンプルコード。


- サーバーはnode.js, express, Vanilla JS
- import { RealtimeClient } from '@openai/realtime-api-beta';
- ./v2cc.js を参考にできる。これは手元で動いてる。 これはコマンドラインツールのソース。
- WebSocketで音声とSTT結果を送りあう
- 手元のmacbookでサーバを起動する 192.168.11.9みたいなアドレス。
- LAN上につながってるiPhoneでサファリで開いてアプリを体験する。
- クライアントでリアルタイム音量を表示する。
- マイクデバイスを手動選択できるようにする
- .envにOpenAIのキーを保存する
- サーバーで1秒に1回音量を表示する。
- マイクが起動したらconsole.logする
- マイク音量を1秒に1回、期間の最大音量を console.logする
