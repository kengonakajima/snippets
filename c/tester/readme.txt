Tester プログラム群

ネットワーク上のサーバのいろいろな性能を試験するための
プログラム。基本的に負荷をかけることが主な仕事。

* connector : connectしてcloseする というのをたくさん繰りかえす。
* tcp_rw : connectしてread/writeをたくさんする。書きこむ内容はバイナリ、
テキストどちらでも、入力ファイルで指定することができる。バイナリをテキスト
で与えることも可能。バイナリデータをエディタで作れるようにするのが目的。
closeするまえにループする回数も指定できる。タイミングも。
* udp_rw :  UDPで入力ファイル(テキスト)通りのバイナリデータを送信
しまくり。
