

```
+------+  +--------+
|   A  |  | outputs|
+------+  +--------+
+------+  +--------+
|   B  |  |        |
+------+  |timeline|
+------+  |        |
|   C  |  |        |
+------+  +--------+

```

この画面は、プロジェクトの画面です。


Aは、markdownで現在のプロジェクトのGoalが書かれています。
Bは、markdownで現在のプロジェクトのステータスが書かれています。
Cは、markdownでそれぞれの人間のスタッフのステータスが書かれています。ステータスは、取り組んでいる課題とか、最近の成果物、発言など重要なトピックです。

timelineには、プロジェクトの成果物として、画面写真や、コードのDiffなどがredditみたいに紹介され、ツリーがついています。

outputsは、最近の成果物のサムネイルが6個ぐらいリストされています。
